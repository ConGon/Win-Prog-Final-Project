## Setup Instructions

### Initial Setup

When cloning this repository for the first time, you will need to run the scaffold command to generate necessary files. These generated files are excluded from version control via `.gitignore`.

```bash
scaffold
```

### After Each Pull

Due to `.gitignore` configuration that excludes generated files, you must run the scaffold command after each pull from the repository:

```bash
scaffold
```

## Automation

For convenience, consider setting up a command or script to automatically run the scaffold command, which will streamline the development workflow and reduce manual setup steps.
